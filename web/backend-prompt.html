راهنمای ساخت بک‌اند — برای Claude Code

این متن، شرح کامل بک‌اندِ «صندوق مهر۱۱۰» است: همه‌چیز روی Firebase میزبانی می‌شود (وب + بک‌اند)، و تمام تعامل‌ها فقط از طریق API انجام می‌شوند. متن داخل کادر را کپی کنید و به Claude Code بدهید.

پرامپت (انگلیسی — برای ایجنت)

نکته: این پرامپت انگلیسی نوشته شده تا برای ایجنت برنامه‌نویس دقیق و بی‌ابهام باشد — همهٔ روابطِ فیلدهای محاسباتی، قوانین تجاری، و الزام «همه‌چیز از طریق API روی Firebase» در آن آمده است.
